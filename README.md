# calendar-app-web

react+typescript などを使用した calendar アプリ

## contents

1. React
2. TypeScript
3. Vite
4. Eslint
5. Prettier
6. Storybook
7. Vitest (単体テスト)
8. Material-UI (MUI)

## Usage

```bash
pnpm start
```

## library

| Library                                   | Function / Purpose                        |
| :---------------------------------------- | :---------------------------------------- |
| React                                     | UIコンポーネントライブラリ                |
| date-fns                                  | 日付操作ユーティリティライブラリ          |
| Material-UI (MUI)                         | UIコンポーネントフレームワーク            |
| Emotion (@emotion/react, @emotion/styled) | CSS-in-JSライブラリ (MUIの依存)           |
| Vitest                                    | 単体テストフレームワーク                  |
| @testing-library/react                    | Reactコンポーネントのテストユーティリティ |
| @testing-library/jest-dom                 | Jest DOM用のカスタムマッチャー            |
| Storybook                                 | UIコンポーネントのカタログおよび開発環境  |
| Vite                                      | 高速なビルドツールおよび開発サーバー      |
| TypeScript                                | 型付きJavaScript                          |
| ESLint                                    | コード品質・スタイルチェックツール        |
| Prettier                                  | コードフォーマッター                      |
